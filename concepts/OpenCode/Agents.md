# OpenCode Agents | OpenCode

## 概述 (Overview)
設定和使用專門的代理。代理是針對特定任務和工作流程進行設定的 AI 助手。

## 類型 (Types)
- **主代理 (Primary Agents)**: 直接互動的主要助手（Build, Plan）。
- **子代理 (Subagents)**: 專門執行特定任務的助手（General, Explore, Scout）。

## 設定 (Configuration)
透過 `opencode.json` (JSON) 或 Markdown 檔案定義代理。

## 建立代理 (Creating Agents)
使用 `opencode agent create` 指令。
