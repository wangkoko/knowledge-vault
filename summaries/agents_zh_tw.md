# Summary: OpenCode Agents

## Source
`raw/agents_zh_tw.md`

## Context
This summary covers the different types of agents available in OpenCode.

## Key Points
- **Primary Agents**: Build (default, full tools) and Plan (limited, for planning).
- **Subagents**: General (multi-step tasks), Explore (read-only, codebase search), and Scout (read-only, external research).
- **Configuration**: Can be configured via `opencode.json` or Markdown files in `.opencode/agents/` or `~/.config/opencode/agents/`.
- **Usage**: Switch primary agents with **Tab**; call subagents using **@ mentions**.
